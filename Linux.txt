Extract to linkedIn
1. **ssh**: Used to securely connect to remote servers or machines.
```
ssh user@host
```

2. **scp**: Enables secure file transfer between local and remote systems.
```
scp file.txt user@host:/path/to/destination
```

3. **rsync**: Allows efficient synchronization and backup of files and directories.
```
rsync -avz source/ destination/
```

4. **wget**: Used for downloading files from the web or through HTTP/FTP protocols.
```
wget https://lnkd.in/g-fDHMwW
```

5. **curl**: Enables making HTTP requests and interacting with APIs.
```
curl https://lnkd.in/g8qDdG74
```

6. **top**: Displays real-time system resource usage, including CPU, memory, and processes.
```
top
```

7. **htop**: Similar to `top` but with an enhanced user interface and interactive process viewer.
```
htop
```

8. **df**: Shows disk space usage on file systems.
```
df -h
```

9. **du**: Lists disk usage for files and directories.
```
du -sh /path/to/directory
```

10. **grep**: Searches for patterns in files or output.
```
grep "pattern" file.txt
```

11. **find**: Allows searching for files and directories based on various criteria.
```
find /path/to/search -name "filename"
```

12. **chmod**: Changes file permissions.
```
chmod 644 file.txt
```

13. **chown**: Modifies file ownership.
```
chown user:group file.txt
```

14. **tar**: Creates compressed archives or extracts files from archives.
```
tar -cvzf archive.tar.gz files/
tar -xvzf archive.tar.gz
```

15. **systemctl**: Manages system services and control their behavior.
```
systemctl start service
systemctl stop service
systemctl restart service
```

16. **journalctl**: Views system logs and journal entries.
```
journalctl -u service
```

17. **iptables**: Configures firewall rules and network address translation.
```
iptables -A INPUT -p tcp --dport 80 -j ACCEPT
```

18. **crontab**: Schedules recurring tasks or jobs.
```
crontab -e
```

19. **awk**: Processes text-based data and performs pattern scanning.
```
awk '{print $1}' file.txt
```

20. **sed**: Stream editor for text manipulation.
```
sed 's/foo/bar/' file.txt
